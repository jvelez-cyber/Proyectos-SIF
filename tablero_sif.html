Para que el tablero funcione, configura las reglas en Firebase Console → Firestore → Rules con este código:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /proyectos/{doc} {
      allow read: if true;
      allow write: if true;
    }
  }
}
